# [!DNL Workfront Proof] : les utilisateurs ajoutés à une épreuve ont uniquement un accès en lecture seule

>[!NOTE]
>
>Ce problème a été résolu le 25 juillet 2022.

Lorsque l’utilisateur est ajouté à une épreuve, il bénéficie d’un accès en [!UICONTROL lecture seule] uniquement. Cela touche les rôles suivants :

* [!UICONTROL Modérateur]
* [!UICONTROL Réviseur]
* [!UICONTROL Approbateur]

_Premier signalement le 1 juin 2022._
