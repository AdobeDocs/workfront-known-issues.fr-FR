# Panoramas : impossible de modifier la couleur de la balise

Lorsqu’un utilisateur tente de modifier la couleur d’une balise existante dans la variable [!UICONTROL Gestionnaire de balises], l’utilisateur ne peut pas sélectionner une autre couleur dans la liste déroulante. Lorsque l’utilisateur clique sur une couleur, le clic est inefficace.

Ce problème fait référence à l’attribution de couleurs aux balises dans la variable [!UICONTROL Gestionnaire de balises]. Les balises existantes, y compris leurs couleurs, peuvent être affectées aux cartes comme d’habitude.

_Premier signalement le 30 octobre 2023._
