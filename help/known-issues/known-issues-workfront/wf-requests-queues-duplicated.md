# Demandes : les files d’attente de demandes apparaissent plusieurs fois.

>[!NOTE]
>
>* Ce problème a été résolu le 8 janvier 2026.
>* Le problème a été signalé dans l’expérience de rapports héritée.

Lorsqu’une personne commence à créer une demande et qu’elle sélectionne une file d’attente des demandes, celle-ci apparaît plusieurs fois dans la liste déroulante.

_Premier signalement le 30 décembre 2025._
