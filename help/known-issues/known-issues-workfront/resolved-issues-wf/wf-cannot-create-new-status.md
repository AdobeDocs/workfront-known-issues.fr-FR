# [!UICONTROL Configuration] : impossible de créer un nouveau statut

>[!NOTE]
>
>Ce problème a été résolu le 1 août 2022.

Lorsque l’utilisateur tente de créer un nouveau statut, celui-ci n’apparaît dans aucune des listes de statut de la [!UICONTROL Configuration]. Cela se produit lorsque le nouveau statut contient la même clé qu’un statut supprimé.

_Premier signalement le 12 juillet 2022._
