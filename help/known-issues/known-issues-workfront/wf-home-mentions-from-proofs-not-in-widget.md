# Accueil : les mentions des épreuves n’apparaissent pas dans le widget Mentions

>[!NOTE]
>
>Ce problème a été résolu le 19 février 2026.

Lorsque l’utilisateur est mentionné dans un commentaire sur une épreuve, cette mention n’est pas incluse dans le widget Mentions de l’Accueil.

_Premier signalement le samedi 23 janvier 2026._
