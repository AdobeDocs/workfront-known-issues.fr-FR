# Intégrations : les intégrations de JumpSeat échouent pour les organisations sur Unified Shell

>[!NOTE]
>
>Ce problème a été résolu, mais les utilisateurs doivent reconfigurer JumpSeat comme décrit dans la **solution** ci-dessous.

Lorsque l’organisation d’un utilisateur est migrée vers Adobe UnifiedBecky Shell, ses intégrations JumpSeat existantes ne fonctionnent pas.

**Solution de contournement**

Dans le panneau administrateur JumpSeat, remplacez l’URL de l’application `workfront.com` par `workfront.adobe.com`.

_Premier signalement le jeudi 30 octobre 2024._
