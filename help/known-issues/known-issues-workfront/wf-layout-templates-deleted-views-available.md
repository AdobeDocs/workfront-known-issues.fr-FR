# Modèles de disposition : les vues supprimées sont toujours disponibles

Lorsque l’administrateur supprime une vue d’une liste à l’aide des modèles de mise en page, la vue apparaît toujours dans la liste des vues disponibles pour les utilisateurs. Lorsque cette option est sélectionnée, la vue ne s&#39;applique pas ou applique les colonnes de la vue sans aucune donnée.

_Premier signalement le mercredi 21 janvier 2025._
