# Validations : les validations sont verrouillées automatiquement

Lorsque l’utilisateur tente de prendre une décision pour une épreuve, l’approbation est verrouillée et l’utilisateur ne peut pas prendre de décision. Cela peut se produire même s’il n’y a aucune raison pour que l’épreuve soit verrouillée.

_Premier signalement le mardi 16 février 2026._
