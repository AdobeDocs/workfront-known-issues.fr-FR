# Approbations : impossible d’approuver le document à partir du widget Mes approbations de l’Accueil

Lorsque l’utilisateur tente d’approuver un document à partir du widget Mes approbations de l’Accueil, le document disparaît du widget. Cependant, le document n’est pas approuvé, et si l’utilisateur ou l’utilisatrice actualise la page, le document réapparaît dans le widget.

_Premier signalement le mercredi 10 février 2026._
