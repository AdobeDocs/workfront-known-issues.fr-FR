# Demandes : la vue « Tous » n’affiche pas toutes les demandes

>[!NOTE]
>
>* Ce problème a été signalé dans la nouvelle expérience de demande.
>* Ce problème a été résolu le jeudi 18 février 2026.

Lorsque l’utilisateur consulte la liste des demandes dans la nouvelle expérience de demande et que la vue est définie pour afficher toutes les demandes, seules certaines des demandes s’affichent.

_Premier signalement le mercredi 16 décembre 2025._
