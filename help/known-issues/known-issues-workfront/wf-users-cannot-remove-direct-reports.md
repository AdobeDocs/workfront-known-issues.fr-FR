# Utilisateurs : impossible de supprimer les rapports directs

Lorsque l’utilisateur tente de supprimer des rapports directs d’un utilisateur, il ne peut pas mettre à jour le champ Rapports directs s’il comporte déjà une valeur.

**Solution de contournement**

Utilisez l’une des méthodes suivantes :

* Ajoutez une nouvelle valeur, puis supprimez les valeurs.
* Supprimez le rapport direct de l’organigramme de l’utilisateur.

_Premier signalement le 28 janvier 2026._
