# Validations : liens vers les validations non disponibles

Lorsque l’utilisateur est affecté en tant qu’approbateur à un objet, le lien vers cet objet n’apparaît pas dans la zone Mes mises à jour ou le widget Mes approbations de l’Accueil.

_Premier signalement le mardi 27 octobre 2025._
