# Utilisateurs : les utilisateurs supprimés de Adobe Admin Console sont toujours actifs dans Workfront

Lorsqu’un administrateur supprime un utilisateur dans le Adobe Admin Console, cet utilisateur n’est pas supprimé de Workfront. L’utilisateur est toujours visible et actif dans Workfront.

_Premier signalement le 3 février 2026._
