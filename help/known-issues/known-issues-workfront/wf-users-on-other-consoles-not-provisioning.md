# Utilisateurs et utilisatrices : les utilisateurs et utilisatrices ne sont pas correctement configurés dans l’Adobe Admin Console.

>[!NOTE]
>
>Ce problème a été résolu le mercredi 3 février 2026.

Si une personne existe déjà dans une Adobe Admin Console et qu’elle est ensuite automatiquement configurée dans une console qui inclut Workfront, cette personne n’est pas correctement configurée dans la nouvelle console.

_Premier signalement le 20 novembre 2025._
