# Utilisateurs : les utilisateurs ne sont pas correctement configurés sur le Adobe Admin Console

Si un utilisateur existe déjà sur une Adobe Admin Console et est ensuite automatiquement configuré sur une console qui inclut Workfront, cet utilisateur n’est pas correctement configuré sur la nouvelle console.

_Premier signalement le vendredi 20 novembre 2025._
