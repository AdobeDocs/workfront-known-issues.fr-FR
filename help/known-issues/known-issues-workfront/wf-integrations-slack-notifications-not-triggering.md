# Intégrations : les notifications Slack de Workfront ne se déclenchent pas

Lorsqu’un événement se produit dans Workfront qui doit déclencher une notification Slack, la notification n’est pas envoyée.

Les événements suivants sont concernés :

* Un utilisateur commente une tâche ou un projet affecté à l&#39;utilisateur concerné
* L’utilisateur affecté est mentionné ou identifié dans un commentaire dans Workfront.

Cela peut se produire même si les préférences de notification de l’utilisateur sont configurées pour envoyer la notification dans Slack.

_Premier signalement le mardi 2 février 2026._
