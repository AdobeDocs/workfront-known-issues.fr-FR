# Demandes : le créateur du formulaire de demande n’a pas accès au formulaire.

>[!NOTE]
>
>Ce problème concerne les fonctionnalités disponibles uniquement dans Workfront Planning.

Lorsque l’utilisateur crée un formulaire de demande dans Workfront Planning, puis tente d’y accéder, il n’y parvient pas et le message « Vous n’avez pas accès à ce formulaire » apparaît.

_Premier signalement le jeudi 29 octobre 2025._
