# Documents : le panneau Résumé est vide

Lorsqu’un utilisateur affiche une liste Documents et clique sur un document, le panneau Résumé s’ouvre, mais reste vide. L’utilisateur ne parvient pas à fermer le panneau Summary.

Si l’utilisateur recharge à nouveau la page, le panneau Résumé peut se remplir comme prévu. Toutefois, si l’utilisateur clique sur un autre document, le panneau récapitulatif de ce document est vide.

**Solution de contournement**

Pour afficher temporairement le panneau Résumé, essayez de recharger la page.

_Premier signalement le mardi 13 janvier 2025._
