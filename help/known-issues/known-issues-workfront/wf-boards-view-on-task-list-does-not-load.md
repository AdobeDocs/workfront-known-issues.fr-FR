# Panoramas : le mode Panorama dans la zone [!UICONTROL Tâches] du projet ne se charge pas

>[!NOTE]
>
>Ce problème a été résolu le 12 octobre 2023.

Lorsqu’une personne tente de charger les tâches sur un projet en mode Panorama, les tâches ne se chargent pas. Le mode Liste se charge comme prévu.

_Premier signalement le 10 octobre 2023._
