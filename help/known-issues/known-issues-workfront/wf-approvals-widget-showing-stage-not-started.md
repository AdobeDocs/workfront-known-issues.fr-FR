# Approbations : le widget Mes approbations affiche les épreuves qui ne se trouvent pas à l’étape à laquelle l’utilisateur est affecté

Lorsque l’utilisateur consulte son widget Mes approbations sur l’Accueil, il voit les épreuves pour lesquelles l’utilisateur est affecté à une étape, mais pas l’étape à laquelle l’utilisateur est affecté. Il peut alors être difficile pour l’utilisateur ou l’utilisatrice de voir rapidement quelle action est nécessaire.

_Premier signalement le 16 février 2026._
