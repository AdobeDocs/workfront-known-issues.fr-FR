# Notifications : l’utilisateur ne reçoit pas les notifications attendues

Lorsqu’un événement se produit qui doit envoyer une notification à un utilisateur, celui-ci ne reçoit pas de notification. Cela peut se produire même si le paramètre de notification de l’utilisateur autorise la notification.

Ce problème affecte les notifications in-app et par e-mail.

_Premier signalement le 12 février 2026._
