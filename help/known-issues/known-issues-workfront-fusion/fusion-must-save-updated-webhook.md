# Workfront Fusion : les mises à jour du webhook ne sont pas enregistrées automatiquement.

Lorsqu’une personne apporte une modification à un webhook, par exemple dans le module d’événement Workfront > Regarder, les modifications ne sont pas enregistrées automatiquement.

La personne **doit enregistrer le scénario** pour conserver les modifications apportées au webhook.

**Solution de contournement**

Enregistrez le scénario.

_Premier signalement le vendredi 17 juillet 2025._
