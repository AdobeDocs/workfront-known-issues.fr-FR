# [!DNL Workfront Fusion]: [!DNL Workfront] >[!UICONTROL  Appel API personnalisé] module renvoyant des erreurs 404 ou 422

* [!UICONTROL RuntimeError [422] Paramètre non valide : Valeur d’état de filtrage &quot;CMS&quot;]
